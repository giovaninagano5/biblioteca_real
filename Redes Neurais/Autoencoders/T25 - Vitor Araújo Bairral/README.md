 # A Trilha Acadêmica
Repositório virtual para a tarefa Trilha Acadêmica da disciplina de Redes Neurais do curso de Bacharelado em Ciência e Tecnologia da Ilum Escola de Ciência

Este trabalho tem como objetivo fornecer um material didático para o entendimento de redes neurais do tipo autocodificadores. Autocodificadores são redes neurais não supervisionadas voltadas principalmente para a redução de dimensionalidade de grandes volumes de dados. Este tipo de rede neural pode ser utilizado juntamente com outros modelos de aprendizado de máquina, permitindo diferentes tipos de funcionalidades. 

Neste trabalho, foi dado foco à funcionalidade de diminuição de dimensionalidade e reconstrução de imagens dos autoencoders.

## Estrutura do projeto:
- Autoencoders.ipynb: é o arquivo principal do projeto, contém todas as explicações teóricas e execuções práticas de código desenvolvidas.
- rede_neural.py: é o script que contém as classes de DataModule e Autoencoder, responsáveis pela manipulação de dados e estruturação das redes neurais.
- treinar.py: é o script que instancia e executa as redes neurais. 
- leitor_imagem.py: contém funções úteis para manipulação de imagens.
- Arquivos de imagem .png: são os gráficos das curvas de aprendizado de diferentes modelos treinados previamente.
- Arquivos .pth: contém os pesos e vieses de modelos treinados previamente.
