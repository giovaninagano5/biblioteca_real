# TrilhaAcadêmica
Repositório para a entrega "Trilha Acadêmica" da disciplina de Redes Neurais e algoritmos genéticos do ano de 2026. A entrega aborda o desenvolvimento de um PINN (Physics-Informed Neural Network) para simular uma ponte segundo o modelo de viga de Euler-Bernoulli.

Os arquivos presentes na pasta de dados de otimização devem ser descompactados e deixados na pasta ois são usados para uma análise comparativa das losses em etapas da implementação.
